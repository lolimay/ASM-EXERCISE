# 实验六 子程序设计
## 实验目的
掌握通用子程序的设计方法，学会子程序的功能划分和正确进行子程序的参数传递。
## 实验内容
以子程序调用的形式，实现从键盘输入一个四位的十六进制数（字母为大写字母），将其转化为十进制数输出显示。四位的十六进制数（其中字母为大写），将其转化为二进制数显示输出，采用循环结构实现。
